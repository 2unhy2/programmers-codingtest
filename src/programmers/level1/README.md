## 완주하지 못한 선수
마라톤에 참가한 선수 명단과 완주한 선수 명단이 주어질 때,
완주하지 못한 선수의 이름을 반환하는 문제

조건
- 참가자 수는 최소 1명
- 완주자 수는 참가자보다 항상 1명 적음
- 동명이인이 존재할 수 있음

어려웠던점 : 해시맵을 사용해 본적이 별로 없어서 문제 풀이 방식부터 헷갈렸음. 추후 단순 비교가 아닌 걸 알았음..
접근법
단순 비교 아닌 각 이름의 등장 횟수를 비교해야 함. 

참가자 명단을 순회하며 이름별 등장 횟수를 해시맵에 저장 -> 완주자 명단을 순회하며 해당 이름의 카운트를 1씩 감소 -> 값이1인 이름이 완주하지ㅏ 못한 선수가 됨.
