# 유연근무제 (Programmers Level 1)
i : 직원
j : 날짜
day : 요일

어려웠던 점
1. startday++로 날짜 계산하려 함. 하지만 루프에서 계쏙 바꾸면 헷갈림
day - 이번 날짜가 무슨 요일인지 확인

2. 처음엔 날짜 기준으로 돌아서 번거로웠음. 상품 받는 직원 기준으로 판단하여 한번이라도 실패하면 ok=false로 지정

3. +10은 그냥 더하면 안되는 경우가 있었음. 958에서 +10을하면 1008이 아닌 968이됨.
따라서 아예 분 단위로 바꾸어 처리함(toMinute함수 만들어서 사용)

구조
직원 i 한 명 →
  월~일 확인 →
    토/일이면 패스
    평일이면:
      출근시간 <= (희망시간 + 10분) ?
        아니면 탈락
끝까지 통과하면 상품 받음

시간 복잡도: O(N × 7)
공간 복잡도: O(1)
